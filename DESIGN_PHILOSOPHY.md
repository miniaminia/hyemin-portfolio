# 백혜민 포트폴리오 - 디자인 철학

## 선택된 디자인 접근법: Modern Minimalist with Elegant Typography

### 핵심 원칙

1. **타이포그래피 우선**: 세련된 폰트 계층구조로 정보의 우선순위를 명확히 합니다.
2. **여백의 미학**: 충분한 공간을 활용하여 콘텐츠를 호흡하게 합니다.
3. **미묘한 상호작용**: 스크롤 애니메이션과 호버 효과로 우아한 사용자 경험을 제공합니다.
4. **일관된 브랜딩**: 전체 사이트에서 통일된 시각 언어를 유지합니다.

### 색상 철학

- **배경**: 깨끗한 흰색 (`#FFFFFF`)
- **텍스트**: 깊은 회색 (`#1A1A1A`)
- **악센트**: 밝은 녹색 (`#22C55E`) - 참고 사이트의 활기찬 에너지 반영
- **보조**: 라이트 그레이 (`#F5F5F5`)

### 레이아웃 패러다임

- **비대칭 구조**: 텍스트와 이미지의 균형잡힌 배치
- **섹션 기반**: 각 섹션이 명확한 시각적 경계를 가짐
- **흐름**: 위에서 아래로 자연스러운 정보 흐름

### 시그니처 요소

1. **큰 타이포그래피**: 메인 제목에 대담한 폰트 사이즈 사용
2. **프로필 이미지**: 둥근 모서리와 미묘한 그림자
3. **프로젝트 카드**: 호버 시 미묘한 상승 효과

### 상호작용 철학

- **부드러운 전환**: 모든 상태 변화에 0.3초의 트랜지션 적용
- **명확한 피드백**: 호버 상태에서 시각적 변화 제공
- **스크롤 애니메이션**: 요소들이 스크롤 시 부드럽게 나타남

### 애니메이션 가이드

- **Fade In**: 섹션 진입 시 0.6초 페이드 인
- **Slide Up**: 프로젝트 카드는 아래에서 위로 0.5초 슬라이드
- **Scale on Hover**: 프로젝트 카드 호버 시 1.02배 확대

### 타이포그래피 시스템

- **Display**: Pretendard Bold (48px) - 메인 제목
- **Heading**: Pretendard SemiBold (32px) - 섹션 제목
- **Subheading**: Pretendard Medium (20px) - 소제목
- **Body**: Pretendard Regular (16px) - 본문
- **Caption**: Pretendard Regular (14px) - 캡션

---

**참고**: 이 디자인은 참고 사이트(silly-lolly)의 세련된 미니멀리즘을 반영하면서도, 백혜민의 전문성과 경험을 효과적으로 표현합니다.
